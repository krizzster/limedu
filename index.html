Because this is a completely static frontend project hosted on Vercel without an active backend web server, you cannot save direct uploads straight into your browser stream permanently.

Follow these simple steps to update:
  1. Drop your PDF into your designated folder inside your local directory (`uploads/friendX/`).
  2. Open your updated data.json file and paste a new row listing your file path and date stamp.
  3. Commit and Push the changes through your GitHub Desktop client!
